# AI CSV Reporting System using LangGraph

![Python](https://img.shields.io/badge/Python-3.11-blue)
![LangGraph](https://img.shields.io/badge/LangGraph-AI-green)
![Gemini](https://img.shields.io/badge/Google-Gemini-orange)
![Streamlit](https://img.shields.io/badge/Streamlit-App-red)

This project is an AI powered reporting system that automatically analyzes any CSV dataset and generates a professional business report. It uses LangGraph to orchestrate a multi agent workflow consisting of a Profiler, Planner, Analyst, Visualizer, and Reporter. Google Gemini is used for planning the analysis and generating insights, while Pandas and Matplotlib handle data processing and visualization. The application also provides a simple Streamlit interface where users can upload a CSV file, specify optional analysis instructions, and receive an AI generated report with visualizations. This project was developed as a beginner to intermediate implementation of LangGraph for automated data analysis and reporting.
